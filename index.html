Getting Started

Streamline email delivery with SMTP JavaScript, simplifying sending emails hassle-free using a JavaScript API.

Include JSMailify Library

To get started, include the JSMailify library in your project. You can do this by either linking the external script via CDN or downloading the JavaScript file directly.

Download
HTML
<script src="https://jsmailify.com/mailify.min.js"></script>

Send an Email

Once you've included the JSMailify library, you can start sending emails in just a few lines of code. Here's an example of how to send an email using JavaScript.

Replace the placeholder values with your actual SMTP server, username, password, recipient's email address, sender's email address, subject, and body.

JavaScript 
Mailify.send({
    host: "smtp.example.com",
    username: "your_username@example.com",
    password: "your_app_password",
    port: 587,
    toEmail: "recipient@example.com",
    fromEmail: "you@example.com",
    fromName: "Your Name",
    subject: "This is the subject",
    body: "And this is the body"
}).then(response => {
    if (response.success) {
        console.log('Email sent successfully');
    } else {
        console.error('Email not sent: ', response.error);
    }
})
.catch(error => {
    console.error('Error: ', error.error);
});

SMTP Server Settings

By configuring your project with the correct SMTP settings, you ensure seamless communication between your application and your email provider's servers. This allows you to send emails reliably and securely, no matter which provider you choose.

If you have two-factor authentication enabled for your mail servers, additional settings may be required to establish SMTP connections.

GMail SMTP Configuration

  • SMTP Server smtp.gmail.com
  • Port 587

For users with two-factor authentication enabled in Gmail, you can utilize an app password:

  1. Simply visit https://myaccount.google.com/u/0/apppasswords, where you can generate a unique app password.
  2. Provide a name that helps you remember where you'll use the app password (e.g. mail, smtp), then select 'Generate' to create it.

Yahoo SMTP Configuration

  • SMTP Server smtp.mail.yahoo.com
  • Port 587

For users with two-factor authentication enabled in Yahoo, you can utilize an app password:

  1. Simply visit https://login.yahoo.com/myaccount/security/, where you can generate a unique app password.
  2. Click Generate app password or Generate and manage app passwords.
  3. Click Get Started.
  4. Enter your app's name in the text field.
  5. Click Generate password.

AOL Mail SMTP Configuration

  • SMTP Server smtp.aol.com
  • Port 587

For users with two-factor authentication enabled in AOL Mail, you can utilize an app password:

  1. Simply visit https://login.aol.com/account/security, where you can generate a unique app password.
  2. Click 'Generate app Password' or 'Generate and manage app passwords' .
  3. Provide a name that helps you remember where you'll use the app password (e.g. mail, smtp), then select 'Generate Password' to create it.

iCloud Mail SMTP Configuration

  • SMTP Server smtp.mail.me.com
  • Port 587

For users with two-factor authentication enabled in Gmail, you can utilize an app password:

  1. Sign in to . appleid.apple.com
  2. In the Sign-In and Security section, select App-Specific Passwords.
  3. Select Generate an app-specific password or select the , then follow the steps on your screen.

ProtonMail SMTP Configuration

This feature is only available to Proton for business users with custom domains.

  • SMTP Server smtp.protonmail.ch
  • Port 587

  1. Simply visit https://account.proton.me/u/0/mail/imap-smtp, where you can generate a unique app password.
  2. Click Generate Token
  3. Enter the following details to create a new SMTP token:
      Token Name: Select a name so you’ll remember where this token is used, such as the service or device name.
      Email Address: Select one of your active custom domain addresses to pair with your token. Copy this email address and use it as the username.
  4. Click Generate.
Your SMTP username and SMTP token (password) will be generated. You can now enter them as the username and password of your Proton Account.

  2. Provide a name that helps you remember where you'll use the app password (e.g. mail, smtp), then select 'Generate' to create it.
